Todo Chatbot Frontend has been deployed!

To access the application:

1. Get the NodePort:
   kubectl get svc {{ include "frontend.fullname" . }} -n {{ .Release.Namespace }} -o jsonpath='{.spec.ports[0].nodePort}'

2. Get Minikube IP:
   minikube ip

3. Access the application at:
   http://<minikube-ip>:{{ .Values.service.nodePort }}

4. Or use port forwarding:
   kubectl port-forward svc/{{ include "frontend.fullname" . }} 3000:{{ .Values.service.port }} -n {{ .Release.Namespace }}
   Then access at: http://localhost:3000

5. View logs:
   kubectl logs -l app=frontend -n {{ .Release.Namespace }}
